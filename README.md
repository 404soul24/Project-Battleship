# Battleship

Classic Battleship game — place your fleet and sink the computer's ships.

## Setup

```bash
npm install
```

## Dev

```bash
npm run dev
```

Opens at `http://localhost:8080`. Hard refresh after code changes.

## Build

```bash
npm run build
```

Output in `dist/`.

## Test

```bash
npm test
```

## How to Play

1. Select a ship size, click your grid to place it
2. Click enemy waters to attack
3. Sink all 5 enemy ships to win
